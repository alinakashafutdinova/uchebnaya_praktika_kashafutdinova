# uchebnaya_praktika_kashafutdinova

Учебный веб-сайт приюта для животных, разработанный в рамках учебной (проектно-технологической) практики
по направлению «Информатика и вычислительная техника», профиль «Веб-технологии»
(Московский Политех, кафедра Инфокогнитивных технологий).

## Назначение

Сайт помогает потенциальным владельцам знакомиться с животными, ищущими дом, предоставляет полезный
справочный контент и сервис расчёта нормы корма, а зарегистрированным пользователям — личный кабинет
с разделением ролей (волонтёр и администратор).

## Исполнители

> Группа 257-321 Кашафутдинова Алина Тимуровна 


## Технологии

- HTML5, CSS3 (адаптивная вёрстка, flexbox/grid)
- JavaScript (ES6+, без фреймворков)
- Данные питомцев и статей — в `js/data.js`, рендеринг на стороне клиента
- **Мини-БД на localStorage** (`js/db.js`): коллекции `users`, `pets`, `requests`
- **Авторизация** (`js/auth.js`): регистрация, вход, роли, сессия

## Роли и возможности

| Роль | Возможности |
|------|-------------|
| Посетитель | Просмотр каталога, статей, калькулятор, форма обратной связи |
| Волонтёр | Личный кабинет: подача заявок (забрать/помочь), просмотр и отмена своих заявок |
| Администратор | Управление статьями (CRUD), рассмотрение заявок (одобрить/отклонить) |

## Демо-доступ

- **Администратор:** `admin@lapki.ru` / `admin`
- **Волонтёр:** `volonter@lapki.ru` / `volonter`
- Либо зарегистрируйте нового волонтёра на странице регистрации.

## Структура сайта

| Страница | Файл | Назначение |
|----------|------|------------|
| Главная | `index.html` | Приветствие, подборка питомцев, шаги, статьи |
| Каталог питомцев | `catalog.html` | Список с фильтрацией, сортировкой и пагинацией |
| Страница питомца | `pet.html?id=N` | Детальная информация о животном |
| Статьи | `articles.html` | Список материалов об уходе |
| Страница статьи | `article.html?id=N` | Полный текст с автором, динамический заголовок |
| Обратная связь | `contacts.html` | Форма с валидацией на JavaScript |
| Регистрация | `register.html` | Регистрация волонтёра |
| Вход | `login.html` | Авторизация по e-mail и паролю |
| Личный кабинет | `cabinet.html` | Кабинет волонтёра / администратора (по роли) |
| Ошибка 404 | `404.html` | Страница несуществующего адреса |

## Запуск

Сайт статический — достаточно открыть `index.html` в браузере.
